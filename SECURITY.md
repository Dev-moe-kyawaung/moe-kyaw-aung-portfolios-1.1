# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.1.x   | ✅ |
| < 1.1   | ❌ |

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not open a public GitHub issue**.

Instead, report it privately via:
- 📧 Email: moekyawaung@programmer.net
- 🔐 Alternative: moekyawaung@cybergal.com

Please include:
1. A description of the vulnerability
2. Steps to reproduce
3. Potential impact

I aim to respond within **48 hours** and will credit responsible disclosures in the CHANGELOG unless anonymity is requested.
